<a name="getting-started"></a>
# Getting Started

FizzDoc is currently available for Scala 2.10 and 2.11.

To get started with SBT, simply add the following to your build.sbt file:

    libraryDependencies += "org.fizzdoc" %% "fizzdoc" % "0.4.1"
